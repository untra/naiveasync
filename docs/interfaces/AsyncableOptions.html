Interface AsyncableOptions

lifecycle options. These are stored in the .meta cache. Some options can only be set at lifecycle creation time.

Async Lifecycle's manage a number of aspects of the operation and is configured execution. Some of these aspects are recorded in the meta cache, which are not associated with the redux store but used internally to control naiveasync operations. This selection is contextual from the instance when .meta() is called on the lifecycle.
